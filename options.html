Settings

Create a key in Google Cloud Console → Enable YouTube Data API v3 → Credentials → API key. For personal use, restrict by API and quota. The key is stored locally in your browser.

How it works

The extension listens for visits to youtube.com/watch, youtube.com/shorts, or youtu.be/… in the address bar. It extracts only the video ID from the URL (no page scraping), then calls the official YouTube Data API to fetch title, description, and duration. Entries are stored locally.
